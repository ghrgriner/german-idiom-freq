The `Redewendung` column in input/dewk_redewendungen_and_regex.txt
is an unmodified list of all entries in the 'Redewendung' category on
German Wiktionary retrieved June 18, 2025:
   https://de.wiktionary.org/wiki/Kategorie:Redewendung_(Deutsch)
This list is made available by German Wiktionary under the license
"Lizenz Creative Commons Namensnennung – Weitergabe unter gleichen Bedingungen 4.0 Unported",
(https://creativecommons.org/licenses/by-sa/4.0/deed.de).

The `link` column in the same input file is a transformation of the
`Redewendung` column into a URL according to Wiktionary conventions.
Each link links to the Wiktionary page for the idiom. From there,
a link to a page with authorship history is available.

--------------

Creation of the output file uses four German corpora from
Universität Leipzig available for download at:
    https://wortschatz.uni-leipzig.de/en/download/German.

The corpora used are:
- Mixed-typical 2011 1 Million Records
- News 2015 1 Million Records
- Wikipedia 2021 1 Million Records
- News 2023 1 Million Records

The corpora were not modified before being processed to generate the
counts of matches but the resulting counts were sometimes then corrected
to obtain the frequency estimate of each idiom. (See README.md and other
project documentation for details.)
